# App::financeta

**App::financeta** or **financeta** (pronounced: _phy-nan-see-ta_), is a graphical user interface (GUI) to
help perform [Technical Analysis](https://en.wikipedia.org/wiki/Technical_analysis)
and backtesting on stocks.
